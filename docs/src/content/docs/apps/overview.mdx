Hands lets you build complete data applications using the stdlib. Your apps can include dashboards for visualization, API routes for data access, and scheduled jobs for automation.

## App types

| Type | Purpose | Example |
|------|---------|---------|
| **Dashboard** | Visual data exploration | Real-time metrics, charts |
| **API Route** | HTTP endpoints | REST API, webhooks |
| **Scheduled Job** | Timed automation | Daily reports, data sync |
| **Monitor** | Background health checks | Uptime monitoring |

## Project structure

A typical app structure:

```
my-workbook/
├── hands.config.ts
├── src/
│   ├── index.ts           # Main dashboard
│   ├── dashboards/
│   │   ├── sales.ts       # Sales dashboard
│   │   └── users.ts       # User analytics
│   ├── api/
│   │   ├── index.ts       # API entry point
│   │   └── users.ts       # User endpoints
│   └── jobs/
│       ├── daily.ts       # Daily tasks
│       └── sync.ts        # Data sync
```

## The stdlib

Import everything you need from `@hands/stdlib`:

```typescript
import {
  sql,           // Database queries
  dashboard,     // Dashboard definitions
  monitor,       // Background monitoring
  integration,   // External connections
  defineConfig   // Configuration
} from '@hands/stdlib';
```

## Quick examples

### Dashboard

```typescript
import { dashboard } from '@hands/stdlib';

export default dashboard({
  title: 'Sales Overview',
  charts: [
    {
      title: 'Revenue by Month',
      type: 'bar',
      query: `
        SELECT
          TO_CHAR(date, 'Mon') as name,
          SUM(amount) as value
        FROM sales
        GROUP BY 1
        ORDER BY MIN(date)
      `
    }
  ]
});
```

### API route

```typescript
import { Hono } from 'hono';
import { sql } from '@hands/stdlib';

const app = new Hono();

app.get('/sales', async (c) => {
  const sales = await sql`SELECT * FROM sales LIMIT 100`;
  return c.json(sales);
});

export default app;
```

### Scheduled job

```typescript
// hands.config.ts
export default defineConfig({
  triggers: {
    scheduled: [
      { cron: '0 9 * * *', handler: './src/jobs/daily.ts' }
    ]
  }
});

// src/jobs/daily.ts
import { sql } from '@hands/stdlib';

export default async function() {
  const report = await sql`
    SELECT COUNT(*) as orders, SUM(total) as revenue
    FROM orders WHERE date = CURRENT_DATE - 1
  `;
  console.log(`Yesterday: ${report[0].orders} orders, $${report[0].revenue}`);
}
```

## Deployment

All app types deploy to Cloudflare Workers:

1. Click "Deploy" in the toolbar
2. Your app is built and uploaded
3. You get a public URL: `https://your-app.workers.dev`

Deployed apps include:
- Dashboard pages
- API routes
- Scheduled triggers (via Cloudflare Cron)
- Static assets
