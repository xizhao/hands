The Hands standard library (`@hands/stdlib`) provides everything you need to build data applications.

## Installation

The stdlib is included automatically in every Hands workbook. Import what you need:

```typescript
import {
  sql,
  dashboard,
  monitor,
  integration,
  defineConfig
} from '@hands/stdlib';
```

## Core exports

| Export | Purpose | Documentation |
|--------|---------|---------------|
| `sql` | Database queries | [SQL Client](/stdlib/sql/) |
| `dashboard` | Dashboard definitions | [Dashboard](/stdlib/dashboard/) |
| `monitor` | Background monitoring | [Monitor](/stdlib/monitor/) |
| `integration` | External connections | [Integration](/stdlib/integration/) |

## Quick examples

### Query data

```typescript
import { sql } from '@hands/stdlib';

const users = await sql`SELECT * FROM users WHERE active = true`;
```

### Create a dashboard

```typescript
import { dashboard } from '@hands/stdlib';

export default dashboard({
  title: 'Analytics',
  charts: [
    {
      title: 'Users by Day',
      type: 'line',
      query: 'SELECT date, COUNT(*) as value FROM users GROUP BY date'
    }
  ]
});
```

### Monitor data

```typescript
import { monitor } from '@hands/stdlib';

export default monitor({
  name: 'low-inventory',
  query: 'SELECT * FROM products WHERE stock < 10',
  schedule: '0 * * * *',  // Every hour
  alert: {
    when: (rows) => rows.length > 0,
    message: (rows) => `${rows.length} products low on stock`
  }
});
```

### Connect to external database

```typescript
import { integration } from '@hands/stdlib';

const external = integration({
  type: 'postgres',
  connectionString: process.env.DATABASE_URL
});

const users = await external.query('SELECT * FROM users');
```

### Configure your workbook

```typescript
import { defineConfig } from '@hands/stdlib';

export default defineConfig({
  name: 'my-workbook',
  triggers: {
    scheduled: [
      { cron: '0 9 * * *', handler: './src/jobs/daily.ts' }
    ]
  }
});
```

## Type definitions

The stdlib is fully typed. Import types when needed:

```typescript
import type {
  Dashboard,
  DashboardChart,
  Monitor,
  MonitorAlert,
  Integration,
  Config
} from '@hands/stdlib';
```
