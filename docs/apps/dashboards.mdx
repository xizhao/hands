# Dashboards

Dashboards are web pages that display your data through charts, tables, and metrics. They're built with React and rendered server-side for fast loading.

## Creating a Dashboard

Ask the AI:

```
Create a dashboard showing sales performance
```

Or be specific:

```
Create a dashboard with:
- Total revenue this month as a big number
- Revenue trend as a line chart
- Sales by region as a pie chart
- Recent orders as a table
```

## Dashboard Structure

Dashboards are React components in `src/pages/`:

```tsx
// src/pages/Dashboard.tsx
import { Card, Chart, Table } from "../components";
import { charts } from "../../charts";

export function Dashboard({ data }) {
  return (
    <div className="p-8 space-y-6">
      <h1 className="text-2xl font-bold">Sales Dashboard</h1>

      <div className="grid grid-cols-3 gap-4">
        <Card title="Revenue" value={data.revenue} />
        <Card title="Orders" value={data.orders} />
        <Card title="Customers" value={data.customers} />
      </div>

      <Chart chart={charts.revenueByMonth} data={data.revenueChart} />

      <Table
        columns={["Date", "Customer", "Amount"]}
        rows={data.recentOrders}
      />
    </div>
  );
}
```

## Adding Charts

Charts are defined in `charts/index.ts`:

```typescript
export const charts = {
  revenueByMonth: {
    id: "revenue-by-month",
    title: "Monthly Revenue",
    type: "line",
    query: `
      SELECT
        date_trunc('month', order_date) as date,
        SUM(amount) as value
      FROM orders
      WHERE order_date > NOW() - INTERVAL '12 months'
      GROUP BY 1
      ORDER BY 1
    `
  },

  salesByRegion: {
    id: "sales-by-region",
    title: "Sales by Region",
    type: "pie",
    query: `
      SELECT region as name, SUM(amount) as value
      FROM orders
      GROUP BY region
    `
  }
};
```

## Data Loading

Data is loaded server-side before rendering:

```tsx
// src/index.tsx
app.get("/", async (c) => {
  const sql = createDb(c.env.DATABASE_URL);

  const data = {
    revenue: await sql`SELECT SUM(amount) FROM orders`,
    revenueChart: await sql`...`,
    recentOrders: await sql`...`
  };

  return c.html(render(<Dashboard data={data} />));
});
```
