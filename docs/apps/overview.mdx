# Building Apps

Hands apps are full-stack data applications that combine:

- **Dashboards** - Interactive web pages with charts and metrics
- **Charts** - Data visualizations powered by SQL queries
- **API Routes** - HTTP endpoints for your data
- **Scheduled Jobs** - Automated data pipelines

All of this runs on Cloudflare Workers at the edge.

## App Architecture

```
┌─────────────────────────────────────────────┐
│                 Your App                    │
├─────────────────────────────────────────────┤
│  Dashboards (React SSR)                     │
│  ├── Charts (SQL queries)                   │
│  ├── Tables                                 │
│  └── Metrics                                │
├─────────────────────────────────────────────┤
│  API Routes (Hono)                          │
│  ├── GET /api/data                          │
│  ├── POST /api/webhook                      │
│  └── Custom endpoints                       │
├─────────────────────────────────────────────┤
│  Scheduled Jobs (Cron)                      │
│  ├── Data pipelines                         │
│  ├── Reports                                │
│  └── Sync tasks                             │
├─────────────────────────────────────────────┤
│  PostgreSQL Database                        │
│  (Embedded locally, Hyperdrive in prod)     │
└─────────────────────────────────────────────┘
```

## Quick Example

Create a complete app with one prompt:

```
Create a sales dashboard with:
- Total revenue card
- Revenue by month line chart
- Top products table
- An API endpoint to get current metrics
```

The AI generates:

1. Dashboard page with React components
2. Chart definitions with SQL queries
3. API route returning JSON
4. All the styling and configuration

## Tech Stack

Your apps use production-ready technologies:

| Component | Technology |
|-----------|------------|
| Runtime | Cloudflare Workers |
| Web Framework | Hono |
| UI | React 19 + TailwindCSS |
| Rendering | Server-Side Rendering (SSR) |
| Database | PostgreSQL |

## File Structure

```
my-workbook/
├── src/
│   ├── index.tsx          # Main entry point
│   ├── pages/             # Dashboard pages
│   │   └── Dashboard.tsx
│   ├── components/        # Reusable UI
│   └── lib/
│       └── db.ts          # Database utilities
├── charts/
│   └── index.ts           # Chart definitions
├── monitors/              # Scheduled jobs
├── wrangler.toml          # Cloudflare config
└── hands.config.ts        # Hands config
```
