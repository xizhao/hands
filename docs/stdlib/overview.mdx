# Standard Library

The Hands stdlib provides the building blocks for data applications. Import from `@hands/stdlib` to use these in your code.

## Runtime Helpers

Use these in your handler files:

```typescript
import { monitor, dashboard, integration, sql } from "@hands/stdlib";
```

## Quick Reference

### Monitor (Scheduled Job)

```typescript
// monitors/cleanup.ts
import { monitor } from "@hands/stdlib";

export const handler = monitor(async (ctx) => {
  await ctx.sql`DELETE FROM logs WHERE age > 30`;
  ctx.log("Cleanup complete");
  return { status: "ok" };
});
```

### Dashboard

```typescript
// dashboards/main.ts
import { dashboard } from "@hands/stdlib";

export const handler = dashboard(async (ctx) => {
  const stats = await ctx.sql`SELECT COUNT(*) FROM users`;
  return {
    title: "Dashboard",
    widgets: [
      { type: "metric", label: "Users", value: stats[0].count }
    ]
  };
});
```

### Integration

```typescript
// integrations/webhook.ts
import { integration } from "@hands/stdlib";

export const handler = integration(async (ctx) => {
  const data = ctx.request.body;
  await ctx.sql`INSERT INTO events ${ctx.sql(data)}`;
  return { status: 200, body: { received: true } };
});
```

### Configuration

```typescript
// hands.config.ts
import { defineConfig } from "@hands/stdlib";

export default defineConfig({
  name: "my-app",

  monitors: {
    cleanup: {
      schedule: "rate(1 day)",
      handler: "monitors/cleanup.handler"
    }
  },

  dashboards: {
    main: {
      title: "Main Dashboard",
      handler: "dashboards/main.handler"
    }
  },

  integrations: {
    webhook: {
      type: "webhook",
      handler: "integrations/webhook.handler"
    }
  }
});
```

## Handler Context

All handlers receive a context object with:

| Property | Type | Description |
|----------|------|-------------|
| `sql` | `SqlClient` | Database query client |
| `log` | `(msg: string) => void` | Structured logging |
| `env` | `Record<string, string>` | Environment variables |

Additional properties vary by handler type.

## Type Safety

The stdlib is fully typed. Use TypeScript for the best experience:

```typescript
import { monitor, MonitorContext, MonitorResult } from "@hands/stdlib";

export const handler = monitor(async (ctx: MonitorContext): Promise<MonitorResult> => {
  // Full type checking and autocomplete
  return { status: "ok" };
});
```
