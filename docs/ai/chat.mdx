# AI Chat

The AI chat is your primary interface for building with Hands. Ask questions, request features, and get help‚Äîall in natural language.

## How It Works

Type what you want in the input field. The AI:

1. Understands your request
2. Plans the necessary steps
3. Executes code, SQL, and file changes
4. Shows you the results

```
You: Create a chart showing revenue by month

AI: I'll create a line chart for monthly revenue.

[Writes chart definition to charts/index.ts]
[Adds chart to dashboard page]

Done! The chart is now visible at http://localhost:8787
```

## What You Can Ask For

### Data Questions

```
Show me the top 10 customers
What's our average order value?
How many orders did we get last week?
Find duplicate entries in the users table
```

### Building Features

```
Create a dashboard for sales metrics
Add an API endpoint for customer data
Make a scheduled job to clean old records
Build a webhook handler for Stripe events
```

### Code Changes

```
Add a date filter to the dashboard
Make the table sortable
Change the chart color to blue
Rename the 'amount' column to 'revenue'
```

### Debugging

```
Why is this query slow?
The chart isn't loading - help me fix it
I'm getting a TypeScript error here
```

## Sessions

Each conversation is a **session**. Sessions:

- Maintain context as you work
- Remember what you've discussed
- Track the AI's changes

Start a new session for unrelated tasks:
- Click the + button in the toolbar
- Use `Cmd/Ctrl + T`

Switch between sessions using the chips at the top of the thread.
