## Permissions

The AI asks permission before:

- Writing to new files
- Running potentially dangerous commands
- Modifying sensitive data

You can respond:

- **Allow Once** - Permit this specific action
- **Always Allow** - Remember this permission
- **Reject** - Deny the action

## Configuring Tool Access

For agents, control tool access in the YAML frontmatter:

```yaml
tools:
  read: true
  write: true
  edit: true
  bash: true
  sql: true
  webfetch: false
  websearch: false
```

And fine-grained permissions:

```yaml
permission:
  bash:
    "npm install *": "allow"
    "rm -rf *": "deny"
  write:
    "src/*": "allow"
    "*.config.js": "deny"
```

## Custom Tools

Create custom tools in `.opencode/tool/`:

```typescript
// .opencode/tool/custom-analysis.ts
import { defineTool } from "@opencode-ai/sdk";

export default defineTool({
  name: "custom_analysis",
  description: "Run custom analysis on data",
  parameters: {
    type: "object",
    properties: {
      query: { type: "string" }
    },
    required: ["query"]
  },
  execute: async ({ query }) => {
    // Custom logic here
    return { result: "analysis complete" };
  }
});
```
