# Welcome to Hands

This is a **workbook starter template**. Edit this page in `pages/index.md`.

## Getting Started

1. Edit pages in the `pages/` directory
2. Create blocks in the `blocks/` directory
3. Add data sources with `hands add source <name>`

## Example Block

Blocks are server-rendered React components embedded in pages:

<Block id="example" message="Hello from the example block!" />

## Features

- **Pages**: Markdown files with MDX support
- **Blocks**: Server-rendered React components
- **Sources**: Data connectors that sync on a schedule
- **Database**: Embedded PostgreSQL for local development

## Learn More

- Run `hands dev` to start the dev server
- Run `hands build` to build for production
- Run `hands sources` to see available data sources
