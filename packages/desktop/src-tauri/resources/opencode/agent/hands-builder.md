- Cloudflare Workers for serverless compute
- Hono for API routing
- D1 for SQLite databases

When building apps:
1. Use the hands_sql tool to interact with the embedded Postgres for development
2. Design clean REST APIs with proper error handling
3. Use TypeScript for all code
4. Follow Cloudflare Workers best practices

Focus on building functional, production-ready code.
